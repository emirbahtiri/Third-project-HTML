Tables in HTML5

In html we can create tables, they are created with some tags
tag to create a table is tag table it accepts tag tr (table rows) and tag td (table cells).


e.g.

1 2 3
4 5 6
7 8 9

In long tables we can use three other tags that are thead, tbody and tfoot these tags are used to divide the table and beat us with easy stylising with css.
Inside thead we can use tr, td and another tag th this tag can be used instead of td and elements inside table head will be bold automaticly.




On the table we have some attributes that serve to unite the cells, they are: colspan which unites cells between itself and rowspan which unites rows
